## ADDED Requirements

### Requirement: Interfaces Documentation
The `/docs/05-interfaces.md` and `/docs/07-glossario.md` files SHALL document system interfaces and terminologies.

#### Scenario: Understanding Interfaces and Terms
- **WHEN** reading `05-interfaces.md`
- **THEN** the reader understands chat/IA interaction interfaces, network protocols (CHAR_INFO/NPC_INFO), and the admin panel structure.

#### Scenario: Looking up a Term
- **WHEN** reading `07-glossario.md`
- **THEN** technical terminologies such as DNA, DNA-Weight, Zone-Listener, and Trade-Cycle are clearly defined.

### Requirement: Master Ledger
The `/docs/SPEC.md` file SHALL serve as the master ledger for tracking implemented functionality.

#### Scenario: Tracking Progress
- **WHEN** checking project status
- **THEN** `SPEC.md` provides a clear view of the master task list and implementation status for all planned features.
